# PDF CER reference pipeline

작성일: 2026-06-10

CER 정답지는 평가용 gold이므로 PDF 텍스트 추출 결과만 그대로 믿지 않는다.

## 고정 절차

1. PDF 텍스트 레이어를 PyMuPDF로 추출한다.
2. 각 PDF를 PNG로 렌더링해 이미지-only 페이지와 사진 안 글자를 확인한다.
3. 텍스트 레이어가 있는 본문/표는 추출본을 우선 사용한다.
4. 텍스트 레이어가 없거나 사진 안에 글자가 박힌 경우에는 렌더 이미지를 보고 직접 전사한다.
5. 표는 선/칸을 만들지 않고 셀 안 텍스트만 읽는 순서대로 적는다.
6. 페이지 번호, 반복 로고, 반복 머리말/꼬리말은 CER 규칙상 제외할 수 있다.
7. 최종 txt는 UTF-8로 저장하고, 숫자/날짜/금액/고유명사는 화면 표기를 기준으로 확인한다.

## 이번 점검 결과

- `0011`, `0013`, `0014`, `0016`, `0018`: PDF 텍스트 레이어와 현재 정답지의 공백 제외 글자 수가 일치한다.
- `0012`: 이미지-only PDF라 렌더 이미지 기준 전사본을 사용한다.
- `0029`: 사용자가 제공한 PDF 원본 기준으로 재추출해 기존 HWP 내부 추출본을 대체했다.
